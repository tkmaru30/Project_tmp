# 🚀 Slack連携開発ワークスペース

Slackから自然言語で指示するだけで、タスク管理・アイデア管理・文書作成・Webアプリ開発が行える統合開発環境です。

## 📋 プロジェクト概要

### 🎯 主な特徴
- **🤖 Slack完全連携**: チャットで全ての操作が可能
- **📝 統合文書管理**: タスク・アイデア・議事録・メモの一元管理
- **⚡ 自動化ワークフロー**: 文書間の自動連携と変換機能
- **🌐 Webアプリ作成**: 指示するだけでアプリを自動生成
- **🇯🇵 完全日本語対応**: 全ての機能を日本語で利用可能

## 💬 Slackからの指示方法

このワークスペースの全ての機能は、Slackからの自然言語指示で操作できます。以下に主な指示方法を説明します。

### 📋 タスク管理

#### タスクの追加
```
# 基本的なタスク追加
@Cursor 「ユーザー認証システムの実装」タスクを追加してください

# 詳細指定でのタスク追加
@Cursor 緊急タスク「セキュリティパッチの適用」を優先度Criticalで追加してください

# 担当者・期限指定
@Cursor 「API仕様書の更新」タスクを追加してください。期限は来週金曜日、担当は田中です
```

#### タスクの確認・検索
```
# 今日のタスク確認
@Cursor 今日のタスク一覧を教えてください

# 条件指定での検索
@Cursor 緊急タスクの状況を確認してください
@Cursor 完了していないHigh優先度のタスクを表示してください
@Cursor 田中が担当しているタスクを確認してください
```

#### タスクの更新
```
# タスク完了
@Cursor 「ユーザー認証実装」タスクを完了にしてください

# 進捗更新
@Cursor 「データベース最適化」タスクの進捗を50%に更新してください

# ステータス変更
@Cursor 「API開発」タスクのステータスを「Review」に変更してください
```

## 🎨 技術スタック

### フロントエンド（推奨）
- **Next.js 14+** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **React Hook Form**（フォーム処理）
- **Lucide React**（アイコン）

### バックエンド（必要に応じて）
- **Next.js API Routes**
- **Prisma**（データベースORM）
- **JWT**（認証）
- **PostgreSQL / SQLite**

### 開発環境
- **Cursor IDE** + **Background Agent**
- **Slack連携** + **GitHub**
- **Vercel**（デプロイ）

## 🔄 自動化ワークフロー

### 文書間連携
- **アイデア → タスク**: アイデアを実行可能タスクに自動分解
- **議事録 → タスク**: アクションアイテムを自動でタスク化
- **メモ → アイデア/タスク**: メモ内容に応じて適切な形式に変換

### 定期自動実行
```
# 日次実行（毎朝9:00）
@Cursor おはようございます！今日のタスク一覧と緊急事項を教えてください

# 週次実行（毎週月曜日10:00）
@Cursor 先週の完了タスクとアイデアの追加状況をレポートしてください

# 月次実行（毎月1日11:00）
@Cursor 先月の生産性レポートを作成してください
```
