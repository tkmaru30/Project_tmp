# Slack連携システム

アイデア管理とタスク管理をSlackから直接操作できる統合連携システムです。

## 基本的な使用方法

### アイデア管理

#### 新しいアイデアの追加
```
@Cursor 「スマートホーム連携システム」のアイデアをinboxに保存してください
@Cursor アイデア「AIチャットボット」を技術カテゴリで保存してください
```

#### アイデアの検索・確認
```
@Cursor AI関連のアイデアを検索してください
@Cursor 今週追加されたアイデア一覧を表示してください
@Cursor 優先度がHighのアイデアを確認してください
```

#### アイデアの分類・更新
```
@Cursor inbox内の「スマートホーム連携システム」をprojectsフォルダに移動してください
@Cursor 「AIチャットボット」アイデアの優先度をHighに変更してください
@Cursor 「音声認識システム」アイデアのステータスを「planned」に更新してください
```

### タスク管理

#### タスクの追加
```
@Cursor 「ユーザー認証システムの実装」タスクを追加してください
@Cursor 緊急タスク「セキュリティパッチの適用」を優先度Criticalで追加してください
@Cursor 「API仕様書の更新」タスクを追加してください。期限は来週金曜日、担当は田中です
```

#### タスクの確認・検索
```
@Cursor 今日のタスク一覧を教えてください
@Cursor 緊急タスクの状況を確認してください
@Cursor 完了していないHigh優先度のタスクを表示してください
@Cursor 田中が担当しているタスクを確認してください
```

#### タスクの更新
```
@Cursor 「ユーザー認証実装」タスクを完了にしてください
@Cursor 「データベース最適化」タスクの進捗を50%に更新してください
@Cursor 「API開発」タスクのステータスを「Review」に変更してください
```

## 高度な連携機能

### アイデアからタスクへの変換
```
@Cursor ideas/projects/の「音声認識システム」アイデアを実行可能なタスクに分解してください
@Cursor 「AIチャットボット」アイデアを基に開発タスクを作成してください
```

### 自動提案の管理
```
@Cursor 今日の自動提案タスクを確認してください
@Cursor 「React 18へのアップデート」提案を採用してactiveタスクに移動してください
@Cursor パフォーマンス改善の提案を全て確認してください
```

### レポート生成
```
@Cursor 今週のタスク完了レポートを生成してください
@Cursor 月間のアイデア追加状況をレポートしてください
@Cursor プロジェクト進捗レポートを作成してください
```

## 定期的な自動実行

### 日次自動実行 (毎朝9:00)
```
# 今日のタスク確認
@Cursor おはようございます！今日のタスク一覧と緊急事項を教えてください

# 自動提案の確認
@Cursor 新しい自動提案タスクがあれば教えてください

# 期限切れの警告
@Cursor 期限が近いタスクや過ぎているタスクを確認してください
```

### 週次自動実行 (毎週月曜日10:00)
```
# 週次レビュー
@Cursor 先週の完了タスクとアイデアの追加状況をレポートしてください

# バックログ整理
@Cursor バックログのタスクを優先度順に整理してください

# アイデアのレビュー
@Cursor inbox内のアイデアを分類してください
```

### 月次自動実行 (毎月1日11:00)
```
# 月次レポート
@Cursor 先月の生産性レポートを作成してください

# アーカイブ処理
@Cursor 古い完了タスクをアーカイブしてください

# システム最適化提案
@Cursor ワークフロー改善の提案があれば教えてください
```

## 通知設定

### 緊急通知
- Criticalタスクの追加
- 期限切れタスクの発生
- システム障害関連の自動提案

### 日次通知
- 今日の予定タスク
- 新しい自動提案
- 完了タスクのサマリー

### 週次通知
- 週間進捗レポート
- 未完了タスクの警告
- 新しいアイデアのサマリー

## コマンド一覧

### アイデア管理コマンド
| コマンド | 説明 | 例 |
|---------|------|-----|
| `add idea` | 新しいアイデアを追加 | `@Cursor add idea "音声認識システム" to technical` |
| `search ideas` | アイデアを検索 | `@Cursor search ideas by tag "AI"` |
| `move idea` | アイデアを移動 | `@Cursor move idea "音声認識" from inbox to projects` |
| `update idea` | アイデアを更新 | `@Cursor update idea "音声認識" priority to High` |
| `list ideas` | アイデア一覧表示 | `@Cursor list ideas in projects folder` |

### タスク管理コマンド
| コマンド | 説明 | 例 |
|---------|------|-----|
| `add task` | 新しいタスクを追加 | `@Cursor add task "API開発" priority High due 2025-09-25` |
| `complete task` | タスクを完了 | `@Cursor complete task "ユーザー認証実装"` |
| `update task` | タスクを更新 | `@Cursor update task "データベース最適化" status to Review` |
| `list tasks` | タスク一覧表示 | `@Cursor list active tasks` |
| `search tasks` | タスクを検索 | `@Cursor search tasks by assignee "田中"` |

### レポートコマンド
| コマンド | 説明 | 例 |
|---------|------|-----|
| `daily report` | 日次レポート生成 | `@Cursor generate daily report` |
| `weekly report` | 週次レポート生成 | `@Cursor generate weekly report` |
| `progress report` | 進捗レポート生成 | `@Cursor generate progress report for project "音声認識"` |

このSlack連携システムにより、チャットベースでの直感的なアイデア・タスク管理が実現できます。
