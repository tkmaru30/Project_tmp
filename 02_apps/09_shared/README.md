# 09_shared/ - 共通リソース

複数のWebアプリケーションで共通利用するリソースを管理するディレクトリです。

## 推奨構造
```
09_shared/
├── components/     # 共通UIコンポーネント
├── styles/         # 共通スタイル（CSS/SCSS）
├── utils/          # 共通ユーティリティ関数
├── constants/      # 共通定数
├── types/          # 共通型定義（TypeScript）
├── assets/         # 共通アセット（画像、アイコン等）
├── config/         # 共通設定ファイル
└── docs/           # 共通リソースのドキュメント
```

## 管理方針
- 複数のアプリで使用されるもののみを配置
- バージョン管理を適切に行う
- 変更時は影響範囲を確認してから更新
- ドキュメントを充実させる
